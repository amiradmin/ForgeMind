# ForgeMind

ForgeMind is an Industrial Intelligence Platform that integrates:

- Asset Management
- Predictive Maintenance (PdM)
- Energy Optimization
- Computer Vision (YOLO-based inspection)
- CMMS (Maintenance Management System)

## Architecture

- Backend: Django + DRF
- Frontend: React + TypeScript
- AI: PyTorch + YOLO + OpenCV
- Infrastructure: Docker + Redis + PostgreSQL
